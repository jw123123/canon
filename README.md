## 🔴 Canon Renewal Project

> 정보를 보여주는 사이트에서, 사진을 시작하고 싶게 만드는 사이트로

국내 카메라 시장 점유율 1위 브랜드 Canon Korea 메인 페이지를 리디자인한 프로젝트입니다. 기존 사이트의 정보 중심 구조를 개선하고, 제품 탐색과 브랜드 경험을 연결하는 것을 목표로 제작하였습니다.

---

## 📖 프로젝트 개요

### 프로젝트 목적

기존 Canon Korea 홈페이지는 정보 전달에 초점을 맞추고 있어 콘텐츠와 제품 정보가 단순 나열형으로 구성되어 있었고, 이로 인해 사용자의 몰입감과 탐색 효율이 떨어지는 한계가 있었습니다.

이에 사용자가 Canon 브랜드 가치를 자연스럽게 경험하면서 제품을 탐색할 수 있도록 UI/UX를 리디자인하고 HTML/CSS/JavaScript 기반으로 구현하였습니다.

### 프로젝트 목표

- 제품 중심 탐색 UX 개선
- 인터랙션 기반 몰입형 UI 적용
- 직관적인 정보 흐름과 콘텐츠 구조 재정비
- 브랜드 경험 중심의 메인 구조 재설계

### 프로젝트 방식

- 개인 프로젝트 (100%)
- Figma를 활용한 UI 설계
- HTML/CSS/JavaScript 기반 퍼블리싱
- Git을 통한 버전 관리

## ✨ 주요 기능

- GNB Depth2 메뉴 구조 구현
- Hero 영상과 New Product 영역 구현
- Recommend 영역 탭(Camera/Lens/Camcoder/etc) 전환 및 제품 카드 슬라이더
- Canon is Now Hover 카드 너비 확장 인터랙션
- Canon Community 카드 크기/컬러 기반 시각적 위계 구성
- Service 아이콘 카드형 UI 구현
- Footer 사이트맵 및 회사 정보 제공

## 🌱 Learn & Future

### 프로젝트를 통해 배운 점

HTML/CSS/JavaScript와 jQuery 기반 퍼블리싱 환경에서 Slick Slider, Lenis 등 외부 라이브러리를 직접 연동하며 인터랙션을 구현하는 방법을 익혔으며, 사용자 경험을 고려한 UI 설계의 중요성을 배울 수 있었습니다.
